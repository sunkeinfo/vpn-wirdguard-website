客戶查詢與常見問題 (FAQ)

遇到問題了嗎？別擔心，我們在這裏為您解答。大多數常見問題都能在這裡找到解決方案。如果問題依然存在，我們的客服團隊隨時準備為您提供幫助。

連接與性能

連接後沒有網絡，無法上網怎麼辦？

這通常是 DNS 解析問題，是最常見的情況。請嘗試以下步驟：

  1. 編輯設定檔： 打開 WireGuard 客戶端，選擇您的連線並點擊「編輯」。
  2. 修改 DNS： 找到 [Interface] 區塊下的 DNS 這一行。
  3. 更換為公共 DNS： 將其值修改為 1.1.1.1, 8.8.8.8 或 8.8.4.4, 1.0.0.1。
  4. 儲存並重連： 儲存您的修改，然後斷開並重新連接 VPN。
點擊「啟用」後無法成功連接 (無法完成握手)？

無法完成握手 (Handshake) 通常意味著您的設備無法與我們的伺服器建立初始連接。原因可能如下：

  • 網絡防火牆： 您所在的網絡（例如公司、學校或公共 WiFi）可能有防火牆阻止了 WireGuard 使用的 UDP 端口。請嘗試切換到不同的網絡（例如手機熱點）進行測試。
  • 設定檔問題： 請確保您使用的是從我們客戶中心下載的最新設定檔，沒有手動修改過金鑰 (PublicKey) 或端點 (Endpoint) 地址。
連接 VPN 後速度變慢了，如何改善？

VPN 速度受多種因素影響，但您可以嘗試以下方法來優化：

  • 選擇就近節點： 物理距離是影響速度的最大因素。請盡量選擇地理位置離您最近的伺服器節點。
  • 考慮獨享專線： 如果您需要極致的性能和穩定性，不受其他用戶影響，強烈建議您升級到我們的「獨享專線」方案。
  • 嘗試分流工具： 參考我們教程頁面推薦的 TunnlTo 工具，讓其他程式走本地網絡，互不影響。

帳戶與方案

共享節點和獨享專線有什麼區別？

這是我們提供的兩種核心方案，以滿足不同用戶的需求：

  • 共享節點 (Shared)： 您將與其他用戶共享伺服器的資源和 IP 地址。這是最具性價比的選擇，非常適合日常的匿名瀏覽、保障公共 WiFi 安全和訪問一般網站。
  • 獨享專線 (Dedicated)： 您將獲得一個完全為您個人保留的 VPN 隧道和一個**固定的獨享 IP 地址**。這意味著：
    • 性能更穩定： 頻寬僅供您一人使用，不受他人活動影響。
    • IP 純淨度高： 杜絕了因共享 IP 被某些網站（如網上銀行）限制或封鎖的風險。
    • 專業用途首選： 非常適合需要固定 IP 進行遠程辦公、訪問公司內部系統或進行安全線上交易的用戶。
一個設定檔 (帳號) 可以在多少台設備上使用？

您可以在您的所有設備上（電腦、手機、平板）安裝同一個設定檔。

但為了保證最佳的連接穩定性和性能，我們建議**同一個設定檔在同一時間只在一台設備上啟用連接**。

你們是否會記錄我的上網活動 (無日誌政策)？

我們嚴格遵守無日誌政策。 我們絕不會記錄、監控或儲存您的任何網絡流量、瀏覽歷史、DNS 查詢或 IP 地址。您的線上活動完全屬於您自己，我們致力於保護您的絕對私隱。
