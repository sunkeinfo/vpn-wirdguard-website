MyWireGuard 設定指南

第一步：獲取您的 WireGuard 設定檔

在您成功訂閱我們的服務後，您將會在您的客戶中心看到「下載設定檔」的按鈕。每個節點都會對應一個獨立的 `.conf` 設定檔。

請將這個檔案下載到您準備設定的設備上。

第二步：在您的設備上匯入設定檔

請先根據您的操作系統，從我們的「客戶端下載」頁面安裝官方的 WireGuard 客戶端。

  • Windows / macOS： 打開 WireGuard 程式，點擊左下角的「從檔案匯入隧道...」，然後選擇您剛剛下載的 `.conf` 檔案。
  • iOS / Android： 打開 WireGuard App，點擊右下角的「+」號，選擇「從檔案或存檔匯入」，然後找到您的 `.conf` 檔案。

第三步：連接與斷開

設定檔匯入成功後，您會在客戶端的主介面看到一個新的連線項目。

點擊該項目旁邊的開關按鈕即可啟用連接。開關變為綠色，即表示您已成功連接到 MyWireGuard 伺服器。

再次點擊開關即可斷開連接。

Linux 安裝與設定

對於 Debian/Ubuntu 系統，您可以使用以下指令安裝：

sudo apt update
sudo apt install wireguard

安裝完成後，將您的 `.conf` 設定檔複製到 `/etc/wireguard/` 目錄下，然後使用 `wg-quick up [您的設定檔名稱]` 指令來啟動連接。
