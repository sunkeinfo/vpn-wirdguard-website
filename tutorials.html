MyWireGuard 設定指南

基礎設定 (使用官方客戶端)

此方法適用於希望將所有網絡流量通過 VPN 的標準用戶。操作簡單直接。

📄

第一步：獲取設定檔

從您的客戶中心下載所需節點的 .conf 設定檔。

📥

第二步：匯入設定檔

打開官方客戶端，選擇「從檔案匯入隧道...」並選取您的設定檔。

🟢

第三步：連接

點擊連線旁的開關即可啟用，享受安全、私密的網絡瀏覽！

進階推薦：使用 TunnlTo 實現應用程式分流 (更強大)

場景推薦： 您是否希望在玩線上遊戲時使用本地網絡以獲得最低延遲，同時讓您的瀏覽器通過 VPN 保障安全？TunnlTo 正是您需要的強大工具！

什麼是 TunnlTo？為何推薦它？

TunnlTo 是一款基於 WireGuard 的第三方客戶端，它最核心的優勢就是提供了**「應用程式分流」**功能。這意味著您可以精準控制**哪一個應用程式**的流量走 VPN 通道，哪一個走您本地的常規網絡。

⬇️

第一步：下載 TunnlTo

TunnlTo 是一款免費的開源軟件。請從其官網下載並安裝。

前往官網
➕

第二步：匯入設定檔

打開 TunnlTo，點擊「Import tunnel(s) from file」，選擇您的 MyWireGuard .conf 檔案。

🔀

第三步：設定分流

點擊連線設定，進入 "App Routing" 頁面，選擇您希望走 VPN 的應用程式 (如 Chrome)。

✅

第四步：連接並享受

啟用連線！現在只有您選中的 App 會通過 VPN，其他程式則不受影響，實現完美分流。

(伺服器/專家) Linux 安裝與設定

對於 Debian/Ubuntu 系統，您可以使用以下指令安裝：

sudo apt update
sudo apt install wireguard -y

安裝完成後，將您的 .conf 設定檔複製到 /etc/wireguard/ 目錄下 (例如，命名為 mywg0.conf)。

使用以下指令來啟動連接：

sudo wg-quick up mywg0

若要斷開連接，請使用：

sudo wg-quick down mywg0
